Задание 1.3

Найти в интернете описание работы DNS, узнать что такое: ns-сервер, делегирование домена, A-запись, MX-запись.

DNS

Domain Name System — это система доменных имен, которая позволяет преобразовывать доменные имена, понятные человеку (например, example.com), в IP-адреса, понятные машинам (например, 93.184.216.34). Это ключевая технология, которая помогает навигировать в Интернете, делая веб-сайты доступными через удобные для запоминания адреса вместо необходимости запоминать числовые IP-адреса.

NS-сервер

NS-сервер, или сервер имен, — это сервер, который хранит базу данных доменных имен и соответствующих им IP-адресов. Каждый домен должен иметь по крайней мере два NS-сервера, которые обрабатывают запросы к домену и перенаправляют их к соответствующему веб-серверу или другим службам. NS-записи используются для указания этих серверов имен в системе DNS.

Делегирование домена

Делегирование домена — это процесс назначения авторитета за определенный домен определенным NS-серверам. Это означает, что когда запрос отправляется в Интернет с целью получения информации о домене, система DNS направляет этот запрос к назначенным NS-серверам. Делегирование домена позволяет администратору домена контролировать записи DNS, связанные с доменом, указывая, какие серверы имен отвечают за его домен.

A-запись

A-запись, или запись адреса, — это тип записи DNS, который напрямую связывает доменное имя с его IP-адресом (IPv4). Это одна из самых основных и важных записей в DNS, позволяющая пользователям доступ к веб-сайтам через доменные имена. При посещении веб-сайта DNS-сервер ищет A-запись для получения соответствующего IP-адреса, чтобы браузер мог загрузить веб-сайт.

MX-запись

MX-запись, или запись обмена почтой, указывает на сервер, который обрабатывает электронную почту для домена. Это позволяет отправлять и получать электронные письма с использованием адресов, связанных с вашим доменом (например, user@example.com). MX-записи указывают на серверы электронной почты (или почтовые серверы), а также могут указывать приоритет сервера, если их несколько, что позволяет управлять потоком почтового трафика.
